# 快速开始（用户指南）

> 本指南帮助用户快速安装和使用 COC Robot。

## 前置要求

| 软件 | 版本 | 说明 |
|------|------|------|
| Windows | 10/11 | 目前仅支持 Windows |
| Python | 3.10.9 | **必须此版本** |
| 雷电模拟器 | 最新版 | 用于运行游戏 |

## 第一步：环境准备

### 1.1 Windows 显示设置

**重要**：必须设置为 100% 缩放

1. 右键桌面 → 显示设置
2. 缩放与布局 → **100%**

### 1.2 安装 Python

1. 下载 [Python 3.10.9](https://www.python.org/downloads/release/python-3109/)
2. 运行安装程序
3. **务必勾选** "Add Python to PATH"
4. 点击 "Install Now"

验证安装：
```bash
python --version
# 应显示：Python 3.10.9
```

### 1.3 安装 Git

1. 下载 [Git](https://git-scm.com/)
2. 安装过程全部默认即可

## 第二步：安装雷电模拟器

1. 下载 [雷电模拟器](https://www.ldmnq.com/)
2. 安装并启动
3. 设置模拟器：
   - **分辨率**：800 × 600
   - **DPI**：160
   - **性能**：建议 2核 4G内存
4. 关闭所有滤镜插件

## 第三步：安装游戏

1. 在模拟器内安装《部落冲突》国际服
2. 登录游戏账号
3. 进入主世界
4. **使用默认皮肤**（不要使用特殊皮肤）

## 第四步：下载程序

打开命令提示符（Win + R，输入 `cmd`）：

```bash
# 1. 克隆项目
git clone https://github.com/qilishidai/coc_robot.git

# 2. 进入目录
cd coc_robot

# 3. 安装依赖
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple

# 4. 启动程序
python UI入口.py
```

## 第五步：配置机器人

1. 程序启动后，点击右侧 **"配置管理"**
2. 填写配置：
   - **机器人标识**：唯一名称，如 `主号`
   - **模拟器索引**：雷电多开器序号（第一个是 0）
   - **服务器**：国际服
   - **最小资源**：搜索村庄的最低资源要求
3. 点击 **"创建新机器人"**
4. 在左侧选择机器人
5. 点击 **"启动"**

## 推荐配置

### 低本玩家（1-7本）

```
最小资源：100000
边缘比例：0.0
```

### 中本玩家（8-11本）

```
最小资源：200000
边缘比例：0.3
```

### 高本玩家（12本+）

```
最小资源：300000-500000
边缘比例：0.6
```

## 多账号使用

1. 使用雷电多开器创建多个模拟器
2. 每个模拟器登录不同账号
3. 为每个账号创建独立机器人配置
4. 模拟器索引对应多开器序号（从 0 开始）

## 更新程序

```bash
cd coc_robot
git pull
```
